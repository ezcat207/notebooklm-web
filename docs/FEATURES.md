# Notebook and Batch - 完整功能规划

> **定位**: NotebookLM 的网页版 CLI - 提供 CLI 的所有功能，但通过浏览器 UI 操作

参考项目: [ffmpeg-online](https://github.com/xiguaxigua/ffmpeg-online)

---

## 核心理念

将 `nlm` CLI 的所有 35 个工具转化为网页操作界面，让用户无需安装 Python/CLI，只需浏览器即可完成所有操作。

### 主要优势

1. **批量操作** - 批量导入、批量生成、批量查询
2. **功能串联** - 工作流自动化（导入→查询→生成→下载）
3. **自定义 Prompt** - 为每个操作自定义提示词和参数
4. **可视化界面** - 比 CLI 更直观的操作体验

## 核心功能串联示例

### 工作流 1: 竞品研究全流程

```
用户操作流程:

1. [创建笔记本] "竞品分析 - Q2 2026"

2. [批量导入] 输入 10 个竞品 URL
   https://competitor1.com/blog
   https://competitor2.com/product
   ...

3. [AI 查询] "总结所有竞品的核心功能"
   → 获得 AI 总结结果

4. [批量生成]
   ☑ 生成音频 Overview (深度对话, 长)
   ☑ 生成 Infographic (横向, 标准详细)
   ☑ 生成 Slide Deck (详细, 默认长度)

5. [下载管理] 批量下载所有生成的文件

6. [分享] 生成公开链接，分享给团队
```

### 工作流 2: 内容营销自动化

```
1. [创建笔记本] "内容营销 - AI 工具推荐"

2. [研究] Web 研究 "best AI tools 2026"
   → 自动发现 20+ 来源

3. [导入研究结果] 选择前 15 个最相关的

4. [自定义 Prompt 查询]
   "用营销语言总结，突出独特卖点"

5. [批量生成]
   ☑ Report (Blog Post 格式)
   ☑ Infographic (方形, 适合社交媒体)
   ☑ Audio (Brief 格式, 30秒预告)

6. [导出] 导出 Report 到 Google Docs 继续编辑
```

### 工作流 3: 培训材料制作

```
1. [批量创建] 3 个笔记本
   • 新员工入职
   • 产品培训
   • 流程文档

2. [批量添加] 每个笔记本添加对应的文档
   (支持 PDF 上传、Google Drive 同步)

3. [批量生成学习材料]
   ☑ Quiz (每个笔记本 10 题)
   ☑ Flashcards (每个笔记本 20 张)
   ☑ Study Guide Report

4. [批量下载] 所有学习材料

5. [分享给学员] 邀请协作者 (查看权限)
```

## 技术架构

### 前端 (Next.js 14)

```
app/
├── page.tsx                    # 主页 (笔记本列表)
├── notebook/
│   └── [id]/
│       ├── page.tsx            # 笔记本详情页
│       ├── query/              # AI 查询
│       ├── studio/             # Studio 生成
│       └── sources/            # 来源管理
├── batch/
│   └── page.tsx                # 批量操作页
├── workflow/
│   └── page.tsx                # 工作流编排
└── settings/
    └── page.tsx                # 设置页

components/
├── NotebookCard.tsx            # 笔记本卡片
├── BatchImportForm.tsx         # 批量导入表单
├── AIChat.tsx                  # AI 对话组件
├── StudioGenerator.tsx         # Studio 生成器
├── DownloadManager.tsx         # 下载管理器
├── WorkflowBuilder.tsx         # 工作流构建器
└── ProgressIndicator.tsx       # 进度指示器

lib/
├── chrome.ts                   # Chrome API 封装
├── api.ts                      # NotebookLM API 封装
└── types.ts                    # TypeScript 类型
```

```

├── notebooks.js                # 笔记本操作

├── studio.js                   # Studio 操作

```
